# Glow by Diva - Luxury Nail Salon

A sophisticated and minimalist web application for a high-end nail salon, featuring a premium design aesthetic, seamless booking integration, and an interactive gallery.

**Live Demo:** [https://glowbydiva.vercel.app/]

## üåü Features

- **Luxury Minimalist Design**: A "Luxury Minimalist" and "Editorial" aesthetic using a neutral color palette, serif typography, and high-quality imagery.
- **Responsive Layout**: Fully responsive design that looks stunning on mobile, tablet, and desktop devices.
- **Interactive Gallery**: A filterable image gallery showcasing nail art masterpieces with smooth animations.
- **Booking System**: Integrated booking modal for easy appointment scheduling.
- **Service Menu**: Detailed service listings with pricing and descriptions.
- **Testimonials**: Animated testimonial section to build trust and credibility.
- **Direction Aware Hover**: Custom UI components for enhanced user interaction.

## üöÄ Getting Started

Follow these steps to get the project running locally on your machine.

### Prerequisites

- Node.js (v18 or later)
- npm, yarn, pnpm, or bun

### Installation

1.  **Clone the repository:**

    ```bash
    git clone https://github.com/Otuja/glowbydiva.git
    cd glowbydiva
    ```

2.  **Install dependencies:**

    ```bash
    npm install
    # or
    yarn install
    # or
    pnpm install
    ```

3.  **Run the development server:**

    ```bash
    npm run dev
    # or
    yarn dev
    # or
    pnpm dev
    ```

4.  **Open your browser:**

    Navigate to [http://localhost:3000](http://localhost:3000) to view the application.
